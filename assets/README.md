# Assets

Store guide screenshots here.

Primary checklist:
- `SCREENSHOT_CHECKLIST.md`

Recommended workflow:
1. Read `SCREENSHOT_CHECKLIST.md`.
2. Capture screenshots using the exact filenames from the checklist.
3. Replace draft or temporary images with final images using the same names.
4. Re-open `docs/guide.html` and verify every referenced image is present.

Naming rule:
- use lowercase
- use dashes
- keep one stable filename per final screen

Example:
- `student-main.png`
- `teacher-grade-date.png`
- `parent-progress.png`
- `admin-schedule-setup.png`
