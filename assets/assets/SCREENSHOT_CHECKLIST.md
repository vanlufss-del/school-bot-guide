# Чек-лист скриншотов

Это точный список кадров для HTML-инструкции.

Цель:
- снять каждый важный экран один раз
- сохранять файлы под стабильными финальными именами
- не плодить случайные дубли
- сделать кадры читаемыми на телефоне

Общие правила:
1. Открывайте Telegram в чистом чате, где сообщения бота легко читаются.
2. Используйте финальные названия кнопок ровно в том виде, как они видны в боте.
3. Не оставляйте над нужным экраном лишнюю старую переписку.
4. Не кадрируйте слишком тесно, чтобы не терялся контекст.
5. Предпочтительны вертикальные скриншоты.
6. Если на экране есть inline-кнопки, они должны полностью попадать в кадр.
7. Если видны личные ID, частные имена или другие чувствительные данные, замаскируйте их перед публикацией.
8. Сохраняйте каждый файл в `docs/assets/` строго под именем из списка ниже.

Рекомендуемый порядок съемки:
1. Ученик
2. Учитель
3. Родитель
4. Администратор

## Финальная проверка перед публикацией

Перед заменой ссылок в боте проверьте:
1. Каждый файл из этого списка существует в `docs/assets/`.
2. Каждый скриншот читается на узком мобильном экране.
3. На скриншотах нет приватных данных, которые нельзя публиковать.
4. Подписи кнопок совпадают с текущим ботом.
5. После добавления изображений `guide.html` продолжает корректно открываться.
